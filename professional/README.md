# 专业版 Skill

专业版适合 Codex、Claude Code、Cursor、WorkBuddy、ChatGPT Agent 等助手型工具。

它把“追剧式读书改写”变成一个项目工作流：先建立内容大纲，再逐章改写、质检和定向修复。

## 使用方式

1. 把 `professional/` 目录加入你的助手项目。
2. 让助手读取 `SKILL.md` 和 `references/rewrite-rules.md`。
3. 提供书籍目录和正文材料。
4. 要求助手先建立内容大纲。
5. 先试写第一章，通过质量闸门后再继续。

## 推荐启动提示词

```text
请使用 story-reading-rewriter 专业版方法。

我会提供一本难读书的目录和章节正文。
请先不要写正文。
请先完成：
1. 判断正文与非正文材料；
2. 提取全书知识结构；
3. 建立内容大纲；
4. 设计四幕式主线；
5. 输出前 5 章的知识关卡和剧情任务；
6. 标出信息缺口、版权风险和质量风险。
```

## 输出顺序

1. 材料判断。
2. 内容大纲。
3. 章节计划。
4. 第 1 章试写。
5. 质量闸门。
6. 定向修复。
7. 继续下一章。
