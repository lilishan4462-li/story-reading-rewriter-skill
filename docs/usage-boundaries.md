# 使用边界

## 本项目是什么

本项目是一套 AI 辅助改写工作流。

它帮助用户把难读书籍拆成知识关卡，再改写成连续长篇知识小说。

## 本项目不是什么

它不是：

- 书籍上传系统；
- OCR 工具；
- 电子阅读器；
- 云端生成服务；
- 付费会员系统；
- 版权授权服务；
- 法律意见。

## 版权边界

用户应确保自己有权处理输入文本。

推荐使用场景：

- 个人学习；
- 研究和评论；
- 读书会内部材料；
- 教学训练；
- 公版书实验；
- 已授权材料改写。

不推荐的场景：

- 未经授权公开发布受版权保护书籍的整本改写版；
- 用改写规避原书版权；
- 批量生产可能替代原书市场的商业内容。

## 质量边界

AI 输出需要人工判断。

即使章节通过质量闸门，也不代表：

- 概念一定准确；
- 原书观点一定完整；
- 改写没有事实错误；
- 文字可以直接公开出版。

正式使用前，应由用户进行事实核查、版权核查和人工编辑。
