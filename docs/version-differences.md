# 专业版与轻量版区别

## 专业版 Skill

专业版面向会使用 AI 助手的人。

适合：

- Codex；
- Claude Code；
- Cursor；
- WorkBuddy；
- ChatGPT Agent；
- 其他能读取项目文件、维护上下文、执行多步任务的 Agent 工具。

专业版的核心价值是“项目化”：

- 可以把规则、内容大纲、章节计划、成稿和修复记录放在同一个工作区；
- 可以让助手反复读取固定规则；
- 可以围绕一本书长期推进；
- 可以更稳定地保持主角、世界线、时间线和知识线。

专业版适合认真做完整本书，尤其适合需要多轮修改、质量检查、分章推进的用户。

## 轻量版 Lite

轻量版面向只会使用大模型聊天框的人。

适合：

- 豆包；
- DeepSeek；
- Kimi；
- 通义；
- 普通 ChatGPT；
- 其他支持上传文件或粘贴长文本的对话框模型。

轻量版的核心价值是“低门槛”：

- 不要求安装 Skill；
- 不要求会使用代码工作区；
- 只需要上传或粘贴一个规则包；
- 模型会主动主持流程；
- 每轮用项目状态卡降低断上下文风险。

轻量版适合试用、单章实验、轻量读书会材料和小规模改写。

## 如何选择

如果你会使用助手型工具，优先选专业版。

如果你只有聊天框，先用轻量版。

如果你先用轻量版跑通一本书的方向，后续也可以迁移到专业版继续精修。
