# 项目状态卡模板

用于豆包、DeepSeek、Kimi、通义、普通 ChatGPT 等对话框模型的断点续写。

每轮输出末尾都应该保留一张状态卡。

```text
项目状态卡：
- 书名：
- 作者：
- 当前阶段：
- 已处理材料：
- 当前内容大纲版本：
- 主角：
- 主线：
- 世界设定：
- 反派压力：
- 关键配角：
- 已完成章节：
- 当前主角状态：
- 当前未解钩子：
- 已确认知识关卡：
- 已确认设定：
- 禁止改动设定：
- 质量风险：
- 下一步建议：
```

## 使用方式

如果当前对话太长、模型开始遗忘，或你想换平台继续：

1. 复制最近一次状态卡。
2. 新开对话。
3. 上传或粘贴 `story-reading-rewriter-lite.md`。
4. 粘贴状态卡。
5. 要求模型先恢复上下文，再继续下一章。

推荐提示词：

```text
请使用追剧式读书改写器 Lite。
下面是项目状态卡。
请先恢复上下文，复述当前内容大纲和未解钩子。
确认后再继续下一章，不要重开世界线。
```
